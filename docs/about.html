vexstreams.com

vexstreams.com is a website for finding and watching streams of VEX Robotics and other RECF tournaments. Info about events and streams is pulled from RobotEvents.

How do I get my event to show up on this site?

No special action is required – just include information about the stream in the "Webcast" tab on RobotEvents. The "Stream" link shown in the event listing will be the first link included in the "Webcast" tab.

Why doesn't this event's stream work on the multiview page?

Currently, the multiview page only supports events streamed on YouTube or Twitch. Other platforms may be supported in the future, but no promises.

Additionally, if a YouTube stream is set to disallow embeds, it cannot be viewed from the multiview page.
